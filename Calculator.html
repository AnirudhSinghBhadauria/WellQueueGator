• Press "Tab" to Copy Evaluted Result
  • Press "Alt" to Reset the Bar
  • Logarithmic Function can also be Evaluated. // log(10000, 10)
  • Square Root can also be determined. // sqrt(-4)
  • Power Raised to Some Number can also be determined. // pow(2,3)
  • Derivatives can also be found. // derivative('x^2 + x', 'x')
  • Imaginary Functions can be done. // 9 / 3 + 2i
  • Trignometric Angles can also be Found. // sin(45deg)
  • Type Conversions can also be done. // 4 inches to cm
